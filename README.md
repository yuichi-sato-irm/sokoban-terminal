# dirclean - ターミナル倉庫番

> ディレクトリ整理ツールに偽装した倉庫番パズルゲーム（全1000問）

## コンセプト

Claude Code などの AI コーディングツールを使っていると、エージェントがコードを書いている間、エンジニアはただ待つことになります。周囲からは「何もしていない」ように見えがちです。

このゲームは、そんな待ち時間を有効活用するために作りました。

ターミナル上では `dirclean` というディレクトリ整理ツールに見えますが、中身は倉庫番（Sokoban）パズルです。傍から見ると、ファイルの整理作業をしているようにしか見えません。

## 遊び方

```bash
python3 escape.py
```

起動するとメニューが表示されます：

- `n` — 新規ゲーム開始
- `c` — 前回の続きから再開
- `d` — セーブデータを削除
- `q` — 終了

### ルール

- `[]`（ファイル）を `··`（配置先）にすべて押し込めばクリア
- ファイルは押すことしかできません（引けない）
- 壁や他のファイルがある場所には押せません

### 操作

| キー | アクション |
|------|-----------|
| `← ↑ ↓ →` / `h j k l` | 移動 |
| `u` | 1手戻す |
| `r` | レベルやり直し |
| `n` | スキップ |
| `q` | 終了（セーブ＆スコア表示） |

## スコアシステム

各レベルで手数・プッシュ数・クリア時間を計測し、スコアを算出します（低いほど良い）。

- **スコア** = 手数 + プッシュ数 × 2 + 時間ペナルティ
- **ランク**: S > A > B > C > D

終了時にスコアカードとリーダーボードが表示されます。スコアは `.dirclean_scores.json` に保存されるので、ファイルを共有すればエンジニア同士でランキングを競えます。

## セーブ機能

- 進捗は自動でセーブされます（`.dirclean_save.json`）
- 途中でやめても、次回起動時に続きから再開できます
- セーブデータの削除はメニュー画面から `d` キーで行えます

## レベル構成

全1000ステージ、4段階の難易度：

| 難易度 | レベル | ボックス数 | 概要 |
|--------|--------|-----------|------|
| EASY | 1-150 | 1-2 | 基本操作に慣れる |
| MEDIUM | 151-450 | 2-3 | 押す順番を考える必要がある |
| HARD | 451-750 | 2-4 | 回り込みや複数手の先読みが必要 |
| EXPERT | 751-1000 | 3-5 | 本格的なパズル。じっくり考えよう |

全問題は逆再生法（reverse play）で生成し、全問解答可能であることを検証済みです。

## レベルの再生成

```bash
python3 generate_levels.py
```

シード値（デフォルト: 42）を変更すれば、異なる問題セットを生成できます。

## 動作環境

- Python 3.6 以上
- macOS / Linux（ターミナル）
- 外部ライブラリ不要（標準ライブラリのみ）
